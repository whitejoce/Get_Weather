# Get_Weather

* GetWeather.py
 
    Python版本: 3.x

* * *

*   实现方法：

    稍后提供接口的相关信息，供大家自定义天气窗口

    > 通过获取IP定位，爬取当地的天气。
    
    > 脚本中采用的接口不同于网上（网上的的接口天气不更新了），应该是全网首个 ;)
    
    > 天气来源网站[Weather](http://www.weather.com.cn/)；

* * *

*   使用方法:
    
     `~#: python ./GetWeather.py`

* * *
