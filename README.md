# Get_Weather

* GetWeather.py
 
    Python版本: 3.x

* * *

*   实现方法：

    通过获取IP定位，爬取当地的天气
    天气来源网站[Weather](http://www.weather.com.cn/)；

* * *

*   使用方法:
    
     `~#: python ./GetWeather.py`
