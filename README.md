* GetWeather.py
 
     Python版本: 3.x

* * *

*   实现方法：

    > 通过获取IP定位（定位到市级），爬取当地的天气。
    > 
    > 脚本中采用的接口不同于网上（网上的的接口天气不更新了），应该是“全网首发”   ; )
    >
    >定位IP网站[jsonip](http://jsonip.com);
    >
    >IP查询网站[chinaz](http://mip.chinaz.com);
    >
    > 天气来源网站[Weather](http://www.weather.com.cn/)；

* * *

*   使用方法:
    
     `~$: python ./GetWeather.py`
